# Sado Solutions

تطبيق طبي B2B لربط الشركات المصنعة والموزعين للادوية بالصيدليات والمعامل في السودان.

## الميزات الرئيسية

- نظام تسجيل دخول لكل أنواع المستخدمين (شركات، صيدليات، معامل، موردين، إدارة)
- إدارة الطلبيات والمنتجات
- نظام البحث عن المنتجات
- متابعة الطلبيات والتوصيل
- نظام العروض والخصومات
- لوحة تحكم إدارية شاملة
- إشعارات فورية

## المتطلبات

- Flutter SDK >= 3.2.0
- Dart SDK >= 3.2.0
- Firebase Project

## التثبيت

1. استنساخ المشروع:
```bash
git clone <repository-url>
cd sado_solutions
```

2. تثبيت الاعتماديات:
```bash
flutter pub get
```

3. إعداد Firebase:
   - أنشئ مشروع جديد في Firebase Console
   - أضف تطبيق Android و iOS
   - حمّل ملف `google-services.json` و ضعه في `android/app/`
   - حمّل ملف `GoogleService-Info.plist` و ضعه في `ios/Runner/`

4. تشغيل التطبيق:
```bash
flutter run
```

## أنواع المستخدمين

1. **شركة / موزع**: إدارة المنتجات والطلبيات والإحصائيات
2. **صيدلية**: البحث عن المنتجات وعمل الطلبيات ومتابعتها
3. **معمل**: عمل طلبيات الإجراءات المخبرية ومتابعتها
4. **مورد معمل**: إدارة الطلبيات والعروض
5. **إدارة**: لوحة تحكم شاملة لإدارة النظام
